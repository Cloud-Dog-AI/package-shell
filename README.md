# @cloud-dog/shell

Shared application shell layouts, navigation, and framing for Cloud-Dog web applications.

## Install

```bash
npm install @cloud-dog/shell
```

## Example

```tsx
import { AppShell } from '@cloud-dog/shell';

<AppShell appName="Cloud-Dog" navItems={[]}>
  <main>Dashboard</main>
</AppShell>
```
